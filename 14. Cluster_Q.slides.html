何謂分群¶

  • 讓資料說話, 由資料自行分群
    • 將相似的資料歸為一群

何謂相似¶

  • 歐式距離(Euclidean Distance)
  • 曼哈頓距離(Manhattan Distance)
  • 謝比雪夫距離(Chebyshev Distance)

Hierarchical Methods¶

  • Single Link
    • 取最小距離

from scipy.cluster.hierarchy import dendrogram    #引用繪製群圖
from scipy.cluster.hierarchy import linkage       #引用階層分群法
from scipy.cluster.hierarchy import fcluster      #呈現所屬群數


Hierarchical_Variable = linkage(Data, method)  #創建分群

dendrogram(Hierarchical_Variable)  #群圖

fcluster(Hierarchical_Variable, k, criterion='maxclust')    #計算所屬群數

#參數說明
method : 計算群聚的方式(single, complete, average, weighted, centroid, median, ward)
k : 欲分群群數

## 範例

from scipy.cluster.hierarchy import dendrogram, linkage, fcluster
from matplotlib import pyplot as plt
import numpy as np

X = np.array([[42,0.18],
    [36,0.44],
    [18,0.36],
    [50,0.38],
    [38,0.52],])

linked = linkage(X, 'single')

label = range(1, 6)
dendrogram(linked, labels=label)
plt.show()
clusters = fcluster(linked,3,criterion='maxclust')
plt.scatter(X[:,0],X[:,1],c=clusters)
plt.show()
print(clusters)

K-Means¶

K-Means 步驟¶

  • Step 1: 指定預分群群數k
  • Step 2: 於資料中隨機產生k個點
  • Step 3: 計算每一個資料點到k個群心的距離
  • Step 4: 將資料點歸為距離最近的那個群心
  • Step 5: 更新群心（Step 4 這些資料點的平均值作為新的群心）
  • Step 6: 不斷重複Step 3 ~ Step 5直到更新的群心與上一次群心相同

Cluster Evaluate¶

  • 輪廓係數法(Silhouette Coefficient)
    • 群內差異越小, 群外差異越大
    • 越接近1越好，反之越接近-1越差
